# Current Weather API Overview

The Current Weather API provides real-time weather data for any location
worldwide. You can retrieve weather information using different identifiers
such as city name, city ID, geographic coordinates, or ZIP code.

## Available Endpoints

- Get current weather by **city name**
- Get current weather by **city ID**
- Get current weather by **coordinates**
- Get current weather by **ZIP code**

Each endpoint returns a JSON response containing temperature, weather
conditions, wind data, humidity, and more.

:::tip
All Current Weather API endpoints use the same base URL and authentication method.
:::
