# OpenWeatherMap API Documentation

This is a personal project documenting the OpenWeatherMap API using Docusaurus.

## Features
- Authentication guide
- Endpoints with examples
- JSON responses explained
- Error handling
- Tips & common mistakes

## Tech Stack
- Node.js
- Docusaurus
- Postman (for API testing)

## Run Locally
1. `npm install`
2. `npm start`
